# Bmi_calculator
🧮 BMI Calculator (Python)

A simple and user-friendly Body Mass Index (BMI) Calculator built using Python. This program takes user input for weight and height, calculates BMI, and displays the health category.

---

📌 Features

- Accepts user input for weight (kg) and height (meters)
- Validates input (no empty or negative values)
- Handles invalid inputs using exception handling
- Calculates BMI using standard formula
- Displays BMI value with category:
  - Underweight
  - Normal
  - Overweight
  - Obese

---
🧠 Formula Used

BMI = weight / (height²)

---

▶️ How to Run

1. Make sure Python is installed
2. Save the file as "bmi_calculator.py"
3. Run the program:

python bmi_calculator.py

---

💻 Example Output

=== BMI Calculator ===
Enter weight (kg): 60
Enter height (meters): 1.7

✅ BMI: 20.76
✅ Category: Normal
